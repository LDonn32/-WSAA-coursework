# WSAA Coursework 

# ATU Web Services and Applications Module

This repository contains all the lab work, project and assignments from the **Web Services and Applications (WSAA)** module at ATU. This covers fundamental concepts in web services, REST APIs, database integration, and full-stack web development.

##  Repository Overview

- Create and consume RESTful APIs
- Work with HTTP methods (GET, POST, PUT, DELETE)
- Parse and manipulate JSON data
- Interact with third-party APIs
- Design and implement database-driven web applications
- Deploy web applications to cloud platforms
