



## Weekly Lab Work


The `mywork/` folder contains exercises organized by week:


- **Week 01**: Reading/writing files (CSV, JSON, XML), data parsing
- **Week 02**: HTTP requests using `requests` library (GET, DELETE methods)
- **Week 03**: Building REST API clients, data extraction
- **Week 04**: GitHub API exploration, PDF generation
- **Week 05**: Introduction to Flask, basic web servers
- **Week 06**: SQL databases, CRUD operations, data persistence
- **Week 07**: Integration of Flask with frontend components
- **Additional weeks**: Advanced topics and practical exercises
